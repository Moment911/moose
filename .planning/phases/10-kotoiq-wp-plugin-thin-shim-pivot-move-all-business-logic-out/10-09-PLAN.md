<!-- Variable extraction heuristic -->
Walk Elementor JSON tree (depth-first). For each leaf:
- string: if >5 chars + contains alpha → candidate text variable; generate {variable_name}
- url ending in image extension → image variable
- url starting with http(s) → link variable
- Repeater rows: { type: 'list', items: [...] }

The dashboard offers a UI where user names each extracted value.
For v1: auto-generate names from the first few words of the value
(e.g., "Welcome to Acme Plumbing" → {hero_headline}).

<!-- The composeShortcode helper -->

Otherwise: substitute the value directly.

         - Recursive walker:
           ```typescript
           function walk(node: any, path: string, context: string) {
             if (Array.isArray(node)) {
               node.forEach((child, i) => walk(child, `${path}[${i}]`, context));
               return;
             }
             if (node && typeof node === 'object') {
               const element_id = node.id;
               const elType = node.elType || node.widgetType;
               const newContext = elType || context;
               // Settings keys that hold user-facing strings
               if (node.settings && typeof node.settings === 'object') {
                 for (const [key, val] of Object.entries(node.settings)) {
                   if (typeof val === 'string' && isVariableCandidate(val)) {
                     const type = detectType(val);
                     const existing = placeholders.get(val);
                     const varname = existing ?? generateVarName(val, elType, key, placeholders);
                     if (!existing) {
                       placeholders.set(val, varname);
                       variables.push({ name: varname, value: val, path: `${path}.settings.${key}`, type, element_id });
                     }
                     (node.settings as any)[key] = `{${varname}}`;
                   } else if (Array.isArray(val) || (val && typeof val === 'object')) {
                     walk(val, `${path}.settings.${key}`, newContext);
                   }
                 }
               }
               if (node.elements) walk(node.elements, `${path}.elements`, newContext);
             }
           }
           ```
