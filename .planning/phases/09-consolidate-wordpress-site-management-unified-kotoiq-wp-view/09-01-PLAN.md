# Plan 01 — Unified /kotoiq-wp view

## Objective

Build a single `/kotoiq-wp` route that combines fleet ops (today's `/control-center`) and per-client deep dives (today's `/kotoiq-sites` 6 tabs) into one cohesive page. Apply the Unified Marketing brand palette (navy + pink + cream). Redirect `/control-center` and `/kotoiq-sites` to the new route; delete `/wpsimplecode` entirely.

## Must-haves (goal-backward)

1. `/kotoiq-wp` route mounts a working page with a Fleet ↔ Client view toggle.
2. Fleet view shows the same table + stats + bulk-update button as today's `/control-center` (re-styled with Unified palette).
3. Client view shows the same rail + 6-tab pattern as today's `/kotoiq-sites` (re-styled with Unified palette).
4. Clicking a site row in Fleet view auto-switches to Client view with that site pre-selected.
5. `?view=fleet|client` + `?site=<id>` query params deep-link to either state.
6. `/control-center` and `/kotoiq-sites` redirect to `/kotoiq-wp`.
7. `/wpsimplecode` route + view file removed.
8. Sidebar updated: new "KotoIQ WP" link, old WPSimpleCode + Control Center links removed.
9. No regressions to the existing panel components (S&R, Snippets, Access, Builder, Rotation, SEO).
